# Markdown learing

## ✅ Heading

ini adalah contoh heading di Markdown.

# 🖐️ Hai ini heading 1
## 🖐️ Hai ini heading 2
### 🖐️ Hai ini heading 3
#### 🖐️ Hai ini heading 4
##### 🖐️ Hai ini heading 5
###### 🖐️ Hai ini heading 


## ✅ Paragraph

membuat paragraph bisa menggunakan ENTER yang memang sudah dianggap ketika kita mengetikan Enter dan menulis sesuai itu sudah dikatakan paragraph. Pada tag HTML dengan menggunakan <p> </p>

## ✅ Bold
cara menuliskan bold adalah dengan memberikan ** dua bintang yang membungkus sebuah kata atau kalimat dan hasilnya akan seperti tag HTML <b> </b>.
  
Contoh :
**ini adalah tulisan tebal dengan bold**

## ✅ Italic
cara menuliskan bold adalah dengan memberikan * satu bintang yang membungkus sebuah kata atau kalimat dan hasilnya akan seperti tag HTML <i> </i>.
  
Contoh :
*ini adalah tulisan tebal dengan Italic*

## ✅ Blockquote
cara menuliskan bold adalah dengan memberikan > tanda panah kemudian tuliskan kata yang ingin dibuat blockquote.
  
Contoh :
>ini adalah tulisan tebal dengan blockquote
